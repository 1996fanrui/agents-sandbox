Agents Sandbox — Logo Candidates

36 concepts × multiple visual metaphors. Click to view full size.

01 — Shield + Container

盾牌代表安全防护，内嵌终端容器窗口 + 闪电符号代表 Agent 全速运行。

安全力量经典

02 — Hexagon Terminal

六边形（蜂巢/模块化）内嵌终端窗口，虚线环代表隔离边界。紫色系。

科技感终端紫色

03 — 3D Cube in Circle

圆形防护圈 + 3D 立方体沙箱，顶面有 Agent 之眼。冷蓝色调。

3D极简蓝色

04 — Glass Dome + Robot

玻璃穹顶下的机器人 Agent，穹顶网格暗示隔离力场。青绿色系。

趣味具象青绿

05 — Letter Mark "AS"

Agents Sandbox 首字母组合。A 用金色（Agent 活力），S 用蓝色（安全边界）。

字母品牌感双色

06 — Nested Isolation Layers

三层嵌套方框代表宿主→隔离层→沙箱，中心是终端光标。粉色系。四角锁头。

层级感终端粉色

07 — Agent Capsule

胶囊/太空舱造型，Agent 被安全封装其中。电路连接线暗示功能不受限。

未来感可爱青色

08 — Abstract "A" + Sand

字母 A 形成容器结构，横杠是隔离层，下方散落沙粒（sandbox 隐喻）+ 光标。

抽象巧妙紫色

09 — Orbital Shield

Agent 核心被多个轨道环绕保护，轨道节点代表安全检查点。暖橙色。

动感太空橙色

10 — Minimal Terminal "A"

极简终端窗口，大写 A 占据中心，光标闪烁。最干净的开发者风格。

极简开发者绿色

11 — Diamond Fortress

多层菱形嵌套，最内层是终端容器，虚线层代表隔离屏障。钻石=坚固。

几何层次青蓝

12 — Fingerprint Lock

指纹/同心弧线代表多层隔离，中心是锁芯造型。暗示安全认证。

安全抽象紫色

13 — Split Shield

盾牌左右分色：左半金色（Agent 力量/闪电），右半蓝色（安全/锁）。双重性格。

对比盾牌双色

14 — Code Brackets { }

花括号构成沙箱边界，中心是 Agent 之眼。程序员一眼认出的符号。

代码极简绿色

15 — Isometric Layers

等距三层结构：底层 HOST（灰）、中层隔离屏障（金）、顶层 Sandbox（蓝）。

架构等距三色

16 — Container Ship

集装箱码头风格，高亮的活动沙箱容器 + 旗帜。致敬 Docker 的海洋隐喻。

容器Docker紫色

17 — Keyhole Eye

眼睛形状 + 锁孔瞳孔：Agent 的感知力 × 安全锁定。红色系，辨识度极高。

独特安全红色

18 — Circuit Board

芯片/电路板风格，外框连接电路走线，内核标注 AI。硬件安全隐喻。

硬件电路青绿

19 — Portal Gateway

传送门/星际之门，Agent 步入隔离世界。漩涡光效 + 人形剪影。

奇幻传送门紫色

20 — Hourglass Sandbox

沙漏 = sandbox 的字面隐喻。沙粒在封闭空间内流动，底部有终端光标。

双关沙漏金蓝

21 — Maze Boundary

迷宫墙壁构成隔离层，Agent 在核心安全运行。绿色路径标出唯一入口。

迷宫隔离粉色

22 — Atomic Model

原子模型：电子轨道 = 隔离层，原子核 = Agent。经典科学风。

科学轨道青色

23 — Play + Shield

盾牌内大号播放按钮：一键运行，安全无忧。绿色 = Go。

行动盾牌绿色

24 — Infinity Box

无限符号穿透终端窗口：无限能力，有限边界。渐变彩色光带。

无限渐变彩色

25 — Simple Shield Box

纯线条盾牌 + 内嵌终端小窗。无填充、无背景，极简单色。

极简线条蓝色

26 — Double Border

外框=宿主边界，内框=沙箱，间隙=隔离层。虚线连接暗示受控通道。

隔离双层蓝灰

27 — Box + Dashed Ring

实线方框（沙箱）+ 虚线圆环（隔离屏障）。极简双元素。

极简几何紫色

28 — Lock Terminal

锁头的主体就是终端窗口。锁=安全，终端=Agent 运行环境。一体两义。

锁终端绿色

29 — Cube + Lock

线框等距立方体（容器）+ 正面小锁图标。干净的容器隐喻。

3D线条天蓝

30 — Bracket Containment

方括号 [ ] 构成围栏，中间小盒子 + 圆点。代码围栏隐喻。

代码围栏金色

31 — Circle Box Dot

三个几何元素：圆（保护）、方（容器）、点（Agent）。最少笔画。

至简几何紫色

32 — Wall + Window

厚墙中开一扇窗，Agent 在窗内运行。墙体=隔离，窗口=沙箱。

墙壁窗口深蓝

33 — Castle Moat

城堡三层防御：外墙（城垛）+ 护城河（虚线）+ 内堡（沙箱）。

城堡护城河蓝灰

34 — Airlock

气闸门双层密封 + 螺栓 + SEALED 状态。太空站隔离舱隐喻。

密封工业红色

35 — Fence A

字母 A 的横杠变成铁丝网围栏。A=Agent，围栏=隔离。

字母围栏白金

36 — Bubble

气泡包裹终端窗口。高光弧线模拟玻璃质感，轻盈的隔离隐喻。

气泡轻盈青色
